    • Defined in wunderbaum.ts:741

    Apply a modification (or navigation) operation on the tree or active node.

    Most of these commands simply map to a node or tree method. This method is especially useful when implementing keyboard mapping, context menus, or external buttons.

    Valid commands:

    • 'moveUp', 'moveDown'
    • 'indent', 'outdent'
    • 'remove'
    • 'edit', 'addChild', 'addSibling': (reqires ext-edit extension)
    • 'cut', 'copy', 'paste': (use an internal singleton 'clipboard')
    • 'down', 'first', 'last', 'left', 'parent', 'right', 'up': navigate

    Parameters

    • Optional options: ApplyCommandOptions
