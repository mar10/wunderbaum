- Supports drag and drop, editing, filtering, sorting, and multi-selection.
- Written in TypeScript, transpiled to ES6 (esm & umd).
- Performant handling of _big_ data structures.
- Provide an object oriented API.
- Framework agnostic.
- Zero dependencies.
- Keyboard support.
